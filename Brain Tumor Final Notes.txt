3. Intro

This project focuses on automatically identifying brain tumors from MRI images using a deep learning model. These brain tumors are very dangerous so it is important to detect them in early stages. Manual diagnosis through MRI scans is time-consuming and prone to human error. This is where deep learning comes in. By using Convolutional Neural Networks (CNNs), we can automatically analyze MRI images and classify tumors more accurately. Deep learning improves accuracy by automatically learning complex patterns in MRI images. Overall, the goal of my project is to develop a deep learning model that can efficiently detect brain tumors from MRI scans.

5. Design

dataset:
I used a dataset which has around 7000 images of MRI scans with four classes: Glioma, Meningioma, Pituitary tumor, and No Tumor

Preprocessing:
→ The images were resized to match the input size of the pre-trained model
then the images are augmented with rotation, flipping, and zooming to improve generalization. Generalization means like how well a trained model performs on new, unseen data.

Model Selection: (Transfer Learning Approach):
→ Instead of training the model from scratch, i have used a pre-trained CNN model called VGG16. I will go into transfer learning in more detail later on.


Training Process:
→ The model was trained by fine-tuning the last few layers. Fine-tuning is the process of training only some layers of the pre-trained model on the new dataset, while keeping the other layers frozen. Usually we first few layers of the pre-trained model, Because the first layers contain basic features like Detecting edges, lines, and corners, which are useful for any image dataset. We have to freeze to retain these basic features. 

Additionally I extra layers like the Dense and Dropout layers. Dropout layer Prevents overfitting. Overfitting is when a model is learning the training data so well that it includes even noise and other irrelevant details., which makes it perform poorly on new, unseen data.
Dense layer refines the features extracted from the images in dataset. 

Training only the last few layers allows the model to adapt to specific dataset patterns without forgetting pre-trained knowledge. (can read from slide, last sentence of point)


Evaluation:
→ We evaluated the model using accuracy, precision, recall, and F1-score, and visualized the classification results with a confusion matrix.

Precision tells us how many of the tumors predicted as positive are actually correct.
Recall shows how many actual tumors the model successfully detected.
F1-score balances precision and recall to give an overall performance measure


6. Transfer learning

Now let me explain about the working of transfer learning. 

Its a technique in deep learning where a model which is already trained to solve one task is reused or adapted to solve a different, but related task. So instead of training a model from scratch, we use an already pre-trained model. 

As we can see in the diagram, the first model is trained to solve Task A. Suppose Task B is a similar task, then we can use knowledge of the pretrained model to make the new model. So This saves a lot of time. 

Also if you train new model, it is not guaranteed that they will preform well. But if you use pretrained models, the performance will be high as they are already trained on big datasets.
